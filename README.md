# denops-addsymbols
Enclose the word at the cursor position with symbols.

## Requirements
This requires [Deno](https://deno.land) and [denops.vim](https://github.com/vim-denops/denops.vim).

## Usage
### Enclose with ''
```:DSQ```
### Enclose with ""
```:DDQ```
### Enclose with {}
```:DB```
### Enclose with [] 
```:DS```
### Enclose with <>
```:DA```
### Enclose with ()
```:DP```
